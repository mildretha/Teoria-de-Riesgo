# Teoría de Riesgo 📊

Este repositorio contiene materiales, ejercicios y análisis desarrollados durante la asignatura **Teoría de Riesgo**.

## Contenido
- Modelos de riesgo financiero (CAPM, volatilidad, GARCH, EWMA)
- Análisis de mercados financieros
- Ejercicios prácticos en Python/R
- Interpretación de indicadores y métricas de riesgo

## Objetivo
Comprender y aplicar modelos cuantitativos para medir, analizar y gestionar el riesgo en los mercados financieros.

## Herramientas utilizadas
- Python
- R
- Jupyter Notebook
- Librerías de análisis financiero
